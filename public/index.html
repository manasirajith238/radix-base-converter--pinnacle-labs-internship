How it's wired

01

Frontend

Plain HTML, CSS and JavaScript. No frameworks — just fetch() calls to the API as you type, debounced so the backend isn't hit on every keystroke.

02

Backend

A single-file C++17 program opens a raw socket, speaks just enough HTTP to be useful, and does the actual base arithmetic with arbitrary-precision integers — so it isn't capped at 64 bits.

03

The API

GET /api/convert?value=FF&from=16&to=2 returns JSON with the decimal value plus binary, octal, hex, and your chosen base.

Run ./server locally, then open this page — see README for build instructions.
